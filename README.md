## Getting Started

1. Clone the repository:

```
git clone git@github.com:iamlana/timer.git
```

2. Navigate into the directory:

```
cd timer
```

3. Install dependencies:

```
npm install
```

4. Run the application:

```
ng serve
```

5. Open your browser and navigate to http://localhost:4200/.
